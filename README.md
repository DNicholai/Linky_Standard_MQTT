# Linky_Standard_MQTT
# Author - DPegon



Code for ESP32 to interface with French Linky energy meters in Standard Mode & Historic mode using ESP32 & MQTT to transmit the data.  

Components
- 10k resistor ; 
- SFH620A Optocoupler ; 
- ESP32 board ; 
- wiring & perf board ;
